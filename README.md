# CENG 111 Algoritmlar ve Programlama Dersi Raylib Örneği
Bu uygulama Raylib github adresinde bulunan VS Code için oluşturulan şablonu kullanır. İki şubede, ders saati içinde gerçekleştirilen oyunu barındırmaktadır. Raylib kurulum ayarları Intel Mac OS için güncellenmiştir. Orjinal ayarlar için Raylib github adresindeki şablonu kullanınız.

Bu uygulama eğitim amaçlıdır, öğrencilere Raylib kütüphanesinin kullanımını anlatmak için geliştirilmiştir.
